# BreastCancer_Detection
A  Research Paper onBreast Cancer Detection using Tranfer Learning and DeepLearning Methods# 🧠 AI-Based Breast Cancer Detection (Interactive Research Explainer)

## 📌 Overview

This project presents an **interactive web-based visualization** of a research study on breast cancer detection using **transfer learning and deep learning models**.

Instead of a traditional static paper, this platform transforms complex research into a **story-driven, visually engaging experience**, helping users understand:

* The problem of delayed cancer detection
* The role of AI in medical diagnosis
* Model comparison and performance
* Real-world implications of AI in healthcare

---

## 🎯 Objective

To demonstrate how **pre-trained deep learning models** can be applied to medical imaging for accurate cancer detection, and to make the research **accessible, interactive, and intuitive**.

---

## 🚀 Features

* 📖 Story-based research explanation (Problem → Solution → Results)
* 📊 Model comparison dashboard (VGG16, ResNet50, DenseNet121, EfficientNetB0)
* 🧪 Real dataset insights (BUSI & BreakHis)
* 🧠 Deep Learning pipeline visualization
* 🎯 Interactive quiz for learning validation
* 📌 Cheat sheet & glossary for quick revision
* 🎨 Smooth UI with animations and scroll-based transitions

---

## 🏆 Key Results

* Best Model: **EfficientNetB0**
* Accuracy:

  * BUSI Dataset → **89.08%**
  * BreakHis Dataset → **93.75%**
* Achieved high performance with **minimum parameters (5.3M)**
* Demonstrates an efficient and scalable AI solution for healthcare

## 📂 Datasets Used

### BUSI Dataset

* 1,578 ultrasound images
* Classes: Benign, Malignant, Normal

### BreakHis Dataset

* 7,909 histopathology images
* Multiple magnifications

---

## ⚙️ How It Works

1. Image preprocessing (resize + normalization)
2. Data augmentation for training
3. Transfer learning using pretrained models
4. Custom classification head added
5. Training using Adam optimizer
6. Evaluation using accuracy, precision, recall, F1-score
